# Responsive Registration Form

## Objective of the Experiment
The objective of this experiment is to design and develop a fully responsive
registration form using HTML and CSS. The form collects user details and
implements client-side validation using HTML5 attributes to ensure accurate
and valid user input across different devices.

---

## Description of the Project
This project consists of a clean and user-friendly registration form displayed
inside a centered card layout. The form adapts to different screen sizes such
as mobile, tablet, and desktop using responsive design techniques.

---

## Features Implemented
- Responsive layout using CSS
- User-friendly form design
- HTML5 client-side validation
- Input validation using required fields
- Email validation using type="email"
- Password validation using minlength
- Date picker for Date of Birth
- Radio buttons for Gender selection
- Dropdown menu for Country selection
- File upload option for Profile Picture

---

## Technologies Used
- HTML5
- CSS3

---

## Validation Techniques Used
- `required` attribute to prevent empty submission
- `type="email"` for email format validation
- `minlength` for password validation
- `accept="image/*"` for image-only file upload

---

## Responsive Design
- The layout adjusts automatically based on screen size
- Media queries are used to improve mobile responsiveness
- Form elements stack vertically on smaller screens

---

## Challenges Faced
- Maintaining consistent spacing across different screen sizes
- Aligning radio buttons properly for responsiveness
- Ensuring form usability on mobile devices
